# Task report protocol

Русское зеркало основного файла: [`ai/TASK_REPORT_PROTOCOL.md`](../../ai/TASK_REPORT_PROTOCOL.md).

Обязательное правило:

- большие или многоэтапные Issues получают stage result comment после каждого стабильного stage;
- небольшие Issues получают один final result comment перед завершением.
