# Инструкция

Перед изменением прочитайте `ai/README.md`, `ai/HANDOFF_PROTOCOL.md`, `ai/PROJECT_STATE.md`, `ai/DECISIONS.md`, связанный GitHub Issue или PR, текущую branch и `ai/handoffs/INDEX.md`.

GitHub Issues и Pull Requests являются основным контуром.

Файлы `ai/` должны оставаться компактными.
