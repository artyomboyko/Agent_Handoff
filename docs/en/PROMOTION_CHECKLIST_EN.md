# Promotion checklist

This page mirrors `docs/PROMOTION_CHECKLIST.md` for English documentation readers.

## Current release

- Standard 1.3 is active in `AGENT_HANDOFF_STANDARD.md`.
- README, citation metadata, changelog, release notes, project state, and public documentation identify version 1.3.
- Repository checks validate active standard metadata and release-note presence.

## Public repository checks

- Landing README is current.
- Natural search terms are current.
- FAQ and examples are available.
- Contributing, Security, Code of Conduct, and Changelog are available.
- Social preview asset and uploaded social preview image are current.
- About description, Website, and topics are current.
- GitHub Pages is published from `/docs` on `main`.
- Citation metadata matches the active standard version.
- GitHub Actions checks are green.
- Repository history, generated files, Issues, Pull Requests, and Actions logs were reviewed.

## Tagged release

When a tagged GitHub release is planned, create it from the verified merge commit and use the matching file from `docs/releases/` as the release notes source.
