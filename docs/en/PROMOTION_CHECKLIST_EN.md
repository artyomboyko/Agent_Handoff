# Promotion checklist

This page mirrors `docs/PROMOTION_CHECKLIST.md` for English documentation readers.

## Repository presentation

- [x] Landing README explains problem, solution, audience, examples, and comparison.
- [x] Natural search terms are included in README.
- [x] FAQ exists.
- [x] Examples exist.
- [x] Contributing, Security, Code of Conduct, and Changelog exist.
- [x] Social preview asset exists.

## GitHub settings

- [ ] Repository is public when ready.
- [ ] About description is set.
- [ ] Topics are set.
- [ ] Social preview image is uploaded.
- [ ] Release `v1.2` is created.

## Safety

- [ ] Git history reviewed.
- [ ] Secrets are absent or rotated.
- [ ] Issues, PRs, and Actions logs are reviewed.
