# Agent Handoff 1.2

Agent Handoff 1.2 is the first public-ready release of the GitHub-native context handoff standard for AI coding agents, human maintainers, and human-supervised agents.

## Highlights

- Coordinated GitHub Flow for Issue-driven human-agent development.
- Compact repository memory through `ai/` and `ai/handoffs/`.
- Work Claim protocol with agent identity, coordinator, supervision, branch, PR, scope, and status fields.
- Mandatory task result reporting for large multi-stage Issues and small Issues.
- Bilingual English/Russian documentation model.
- Bilingual GitHub Issue Forms and Pull Request template.
- Refactoring workflow and validation/testing workflow.
- GitHub Actions checks for repository structure, YAML forms, PR template checklist, language selectors, docs index parity, and social preview size.
- GitHub Pages landing site.
- `CITATION.cff` metadata for future citation.
- GPL-3.0 license alignment.
