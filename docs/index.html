Context handoff for AI coding agents

Agent Handoff helps AI agents and humans keep durable project context inside GitHub, Git, and compact repository-tracked memory files.

Open repository Read README

Core workflow

Issue
Work Claim
Branch + Draft PR
Checks + Review
Handoff

For humans

See who owns work, what changed, what was tested, what remains risky, and where the next contributor should continue.

For agents

Start from AGENTS.md, claim work in GitHub, open a Draft PR early, keep ai/ compact, and leave a short handoff.

For repositories

Use Issues, PRs, checks, labels, branches, commits, and small durable memory files instead of long chat logs.
