# Handoff Protocol / Протокол handoff

## Start / Старт

Read `AGENTS.md`, `ai/README.md`, `ai/PROJECT_STATE.md`, and the related Issue or PR.
Прочитайте `AGENTS.md`, `ai/README.md`, `ai/PROJECT_STATE.md` и связанное Issue или PR.

## Work / Работа

- Claim the Issue before editing. / Сделайте claim Issue перед изменениями.
- Open a Draft PR early. / Рано откройте Draft PR.
- Keep `ai/` compact. / Держите `ai/` компактным.

## Done / Завершение

- Update PR description. / Обновите PR description.
- Run checks or document why not. / Запустите проверки или объясните почему нет.
- Add handoff when useful. / Добавьте handoff при необходимости.
