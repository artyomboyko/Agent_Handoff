# Code of Conduct / Кодекс поведения

Agent Handoff should be a professional, respectful, and useful project for people working with AI-assisted software development.
Agent Handoff должен быть профессиональным, уважительным и полезным проектом для людей, работающих с AI-assisted software development.

## Expected behavior / Ожидаемое поведение

- Be respectful and constructive. / Будьте уважительны и конструктивны.
- Discuss ideas, not personalities. / Обсуждайте идеи, а не личности.
- Keep Issues and Pull Requests focused. / Держите Issues и Pull Requests сфокусированными.
- Explain risks and trade-offs clearly. / Чётко объясняйте риски и компромиссы.
- Respect maintainer decisions. / Уважайте решения maintainers.

## Unacceptable behavior / Недопустимое поведение

- harassment, threats, or insults / harassment, угрозы или оскорбления;
- spam or repeated off-topic comments / spam или повторяющиеся off-topic comments;
- publishing private data / публикация приватных данных;
- intentionally misleading contributors or agents / намеренное введение contributors или agents в заблуждение.

## Scope / Область действия

This code applies to repository discussions, Issues, Pull Requests, documentation, and project communication.
Этот кодекс применяется к repository discussions, Issues, Pull Requests, документации и коммуникации проекта.

## Reporting / Сообщение о проблемах

For now, report conduct issues to the repository maintainer.
Пока сообщайте о проблемах поведения maintainer репозитория.
