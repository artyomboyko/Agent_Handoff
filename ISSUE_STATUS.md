# Issue Status / Статусы Issue

Status labels / Статусные метки:

- `needs-triage`
- `ready`
- `in-progress`
- `blocked`
- `in-review`
- `changes-requested`
- `ready-to-merge`

Use status labels only while they reflect current state.
Используйте статусные метки только пока они отражают текущее состояние.
