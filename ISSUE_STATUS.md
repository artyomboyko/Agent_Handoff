# AI Handoff Issue Status

Recommended status labels:

| Label | Use for |
|---|---|
| `in-progress` | An agent or maintainer has claimed the Issue and is working on it. |
| `blocked` | Work cannot continue until a dependency, decision, or external condition is resolved. |

Rules:

- Use `in-progress` only together with an Issue claim comment.
- Use `blocked` only together with a short blocker comment.
- Remove stale status labels when work is completed, paused, or handed off.
