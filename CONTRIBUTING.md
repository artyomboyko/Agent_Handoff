# Contributing

Thank you for helping improve Agent Handoff.

## Language

Use English for repository files, Issues, Pull Requests, comments, and documentation in this repository.

Downstream projects may adapt Agent Handoff to another project language, but this repository stays English-only.

## Workflow

1. Open or select a GitHub Issue.
2. Check existing claims and linked PRs.
3. Use a meaningful branch name.
4. Open a Draft PR early.
5. Run checks or document why they were not run.
6. Keep `ai/` files compact.
7. Add a handoff when useful.

## Branch names

Use meaningful names without `/`, Issue numbers, or random identifiers by default.

Examples:

```text
workflow-branch-naming
protocol-work-claim
docs-readme-quickstart
refactor-handoff-index
research-github-flow
```

## Pull requests

A PR should include a related Issue, summary, checks or smoke-test notes, risks, and state updates when needed.
