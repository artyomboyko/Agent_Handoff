# Repository Guide

Before making changes in this repository:

1. Read `AGENT_HANDOFF_STANDARD.md`.
2. Read `ai/README.md`.
3. Read `ai/GITHUB_WORKFLOW.md`.
4. Read `ai/HANDOFF_PROTOCOL.md`.
5. Read `ai/AGENT_IDENTITY.md`.
6. Read `ai/WORK_CLAIM_PROTOCOL.md`.
7. Read `ai/PROJECT_STATE.md`.
8. Read `ai/DECISIONS.md`.
9. Check the related GitHub Issue or Pull Request.
10. Check current branch and recent commits.
11. Check `ai/handoffs/INDEX.md` for relevant active handoffs.

Use GitHub Issues and Pull Requests as the primary workflow system.

Keep `ai/` files small and compact.
